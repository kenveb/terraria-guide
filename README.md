# Terraria Guide

A clean, beginner-friendly Terraria guide website built with static HTML, hosted on Cloudflare Pages.

## Project Structure

```
site/               # Website source code
├── index.html      # Homepage
├── css/            # Stylesheets
├── js/             # JavaScript
├── pages/          # Content pages (guides, builds, etc.)
└── assets/         # Static assets

prd.md              # Product Requirements Document
```

## Tech Stack

- Static HTML5 + CSS3 + Vanilla JS
- Cloudflare Pages (hosting + CDN)
- GitHub (version control)

## Getting Started

1. Clone the repo
2. Open `site/index.html` in browser for local preview
3. Push to GitHub — Cloudflare Pages auto-deploys

## Goal

Validate the "AI-generated content + Cloudflare hosting + ad revenue" model.
